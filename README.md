


### Git common command:
```bash
    git init                                            - ініціалізація локального гіт репозиторія
    git remote add origin <силка на гіт репозиторій>    - добавляє віддалений репозиторій
    git fetch                                           - синхронізує локальні мета дані з гіт сервером (метадані: назви гілок які можна запулити)
    git pull origin <назва гілки>                       - стягує локально все що є на сервері 
    
    git clone <посилання на репозиторій/гілку>          - стягує цілий проект з сервера в окрему папочку
    
    git checkout <назва гілки (develop)>                - локально стаєш на девелоп гілку або ту яку вкажеш
    git add <назви файлів через пробіл>                 - реєструє в локальному репозиторії новий файл для слідкування
    git commit -am'<коміт повідомлення>'                - реєструє коміт повідомлення в локальному репозиторії
    git push origin <назва гілки (develop)>             - відсилає на сервер останні коміт повідомлення і зміни в коді
```
